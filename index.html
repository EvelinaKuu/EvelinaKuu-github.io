Hi, I'm Evelina,

A Sign Language Interpreter turned Web Developer.

After a career as a Sign Language Interpreter and an entrepreneur I went for my goal to become a Web Developer. I put great value on using good online services and apps and wanted the skills to do it myself. The Web Developer Immersive course at General Assembly offered a great start for this. I am currently working as a Front End Developer.

WORK: Project 4

This was my final project on WDI. Due to my interest for fashion, online shopping experiences and vintage/second hand clothing, I created Pre-loved - West London, an app to sell and buy pre-loved fashion in West London. This application is a MERN stack app, has CRUD functionality and also uses JWT/bcrypt authentication. I wanted to expand my knowledge with a new CSS framework and chose Bulma.

Technologies used: JavaScript, React, Express.js, NodeJS, Mocha, Chai, NYC, SASS, MongoDB, Bulma, Axios, Bcrypt
